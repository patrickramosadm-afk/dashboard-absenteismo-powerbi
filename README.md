📊 Dashboard de Absenteísmo em Atendimentos de Saúde
Este projeto foi desenvolvido com o objetivo de analisar padrões de absenteísmo em atendimentos, simulando um cenário real de gestão em um serviço de saúde.

🎯 Objetivo
Identificar padrões de comparecimento, faltas e cancelamentos, além de analisar variações ao longo do tempo e por especialidade.

📌 Principais indicadores
Total de atendimentos
Taxa de comparecimento
Taxa de absenteísmo
Cancelamentos
Análise por especialidade
Tendência temporal

🛠️ Ferramentas utilizadas
Power BI
DAX
Excel (base de dados simulada)
